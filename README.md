# Ejecutar en desarrollo

1. Clonar el repositorio
2. Ejecutar

```
yarn install
```

3. Tener Nest CLI instalado

```
npm i -g @nestjs/cli
```

4. Reconstruir BD con la seed

```
http://localhost:3000/api/v2/seed
```

5. Si se ejecuta el devcontainer solo es necesario ejecutar en consola:

```
   yarn start:dev
```

## Stack usado

- Mongo
- Nest
